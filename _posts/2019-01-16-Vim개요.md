# Vim vs Eamcs --> Vim

Vim(Vi IMproved)은 이름 그대로 vi 에디터의 확장판이다. 1991년 발표된 오래된 프로그램이지만 Emacs와 함께 최고의 에디터 자리를 놓고 경쟁하는 막강한 텍스트 에디터이다. 

텍스트 기반 에디터에 관심이 생긴 것은 키보드를 사면서 부터다. "키보드 끝판왕"을 구입해서 사용하고 있는데, 이 키보드에는 방향키가 없다. 왜 그런가 했더니 키보드 자체가 Emacs 사용에 최적화 됬다고 하는거다. 그래서 Emacs가 뭔지 찾아봤고, 포기했다. 하지만 방향키가 필요없는 에디터를 쓰고싶은 마음에 Vim을 선택해서 IDE로 사용해보려고 한다.

여러 에디터의 Learning Curve이다. Emacs를 왜 포기했는지 설명이 됬으면 한다. 약간의 검색만으로 Emacs를 쓰는 사람들을 존경하게 되었다. 언젠가 시간이 남으면 Emacs도 배워볼 생각.

>  Vim 장점
>
> - 그 어떤 에디터보다도 반응속도 빠르고, 가볍다.
> - 커서의 이동이 매우 빠르다.
> - 아무데서나 다 쓸 수 있다.
> - 잘 쓰는 사람 되면 멋있다.

# Vim의 특징

Vim은 모달(modal) 에디터라고도 불린다. modal은 모드(mode)가 있다는 말이다. 

![image-20190116211649564](https://ws1.sinaimg.cn/large/006tNc79gy1fz8ouw52p7j31j00dcjsi.jpg)

1. Normal mode 
   - 처음 시작할 때, esc를 누르면 노말 모드가 된다.
   - 커서이동, 복사등 간단한 명령 
2. Command mode 
   - 콜론(:)을 누르면 진입한다. 
   - 일반 모드에서는 쉽게 수행할 수 없는 일을 수행할 때 사용.
3. Insert mode 
   - 삽입 모드. i, o, a 키로 진입한다.
4. Replace mode
   - r키로 진입
   - 새로 입력하는 문자로 원래 문자를 대치한다.
5. Visual mode
   - v 키를 눌러 진입한다. 
   - 텍스트를 블럭(block)으로 선택할 수 있음.

# Vim 기본 사용법

가장 빠르게 새로운 프로그램에 적응하는 것은 많이 사용하는 것이다. 하지만 막상 이리저리 사용해 볼 곳이 마땅치가 않을 것이다. Vim을 설치하면 vimtutor라는 명령으로 vim tutorial에 들어갈 수 있다.

직접 Vim을 사용하면서 사용법을 익히게 되어있다. 안에 있는 내용을 간략하게 요약해보겠다.

### 실행과 종료

- 실행 : vim 혹은 vim filename(filename이라는 파일을 만들면서 vim시작)
- 종료 : q는 종료, w는 저장. w, q, q!(저장하지 않고 종료), wq(저장 후 종료) 등으로 사용.
- 파일 열기 : Vim 안에서 :o filename(filename이라는 파일 열기)

### 커서 이동

| h    | 한 칸 왼쪽으로 이동                       | l     | 한 칸 오른쪽으로 이동                 |
| ---- | ----------------------------------------- | ----- | ------------------------------------- |
| j    | 한 줄 아래로 이동                         | k     | 한 줄 위로 이동                       |
| w    | 다음 단어의 첫 글자로 이동                | W     | w와 같은듯                            |
| e    | 다음 단어의 마지막 글자로 이동            | E     | e와 같은듯                            |
| b    | 이전 단어의 첫 글자로 이동.               | B     | b와 같은듯                            |
| ^    | 커서가 위치한 줄의 첫 글자로 이동(글자만) | $     | 커서가 위치한 줄의 마지막 글자로 이동 |
| 0    | 커서가 위치한 줄의 처음으로 이동          | Enter | 다음 줄의 첫 글자로 이동              |
| +    | 다음 줄의 첫 글자로 이동                  | -     | 윗 줄의 첫 글자로 이동                |
| (    | 이전 문장의 첫 글자로 이동                | )     | 다음 문장의 첫 글자로 이동            |
| {    | 이전 문단으로 이동                        | }     | 다음 문단으로 이동                    |
| gg   | 커서 아래로 이동                          | 5gg   | 다섯 칸 아래로 커서 이동              |



### 텍스트 편집 - 삽입 

| i    | 커서가 위치한 곳에서 삽입            | I    | 커서가 있는 줄 맨 앞에서 삽입      |
| ---- | ------------------------------------ | ---- | ---------------------------------- |
| o    | 커서가 위치한 줄 아래에 줄 열고 삽입 | O    | 커서가 위치한 곳 위에 줄 열고 삽입 |
| a    | 커서가 위치한 곳 뒤에 삽입           | A    | 커서가 위치한 줄 맨 뒤에서 삽입    |



### 텍스트 편집 - 지우기

| x    | 커서가 위치한 한 칸 지우기                  |      |                                |
| ---- | ------------------------------------------- | ---- | ------------------------------ |
| dw   | 커서가 위치한 곳 부터 한 단어의 끝까지 지움 | d$   | 커서 위치부터 줄의 끝까지 지움 |
| dd   | 커서가 위치한 한 줄 지움                    | 2dd  | 커서가 위치한 줄 부터 2줄 지움 |



### 텍스트 편집 - 수정

| r     | 커서가 위치한 곳의 한 글자 바꾸기        | R    | esc키를 누를 때 까지 계속 치환       |
| ----- | ---------------------------------------- | ---- | ------------------------------------ |
| cw    | 커서가 위치한 곳 부터 단어의 끝까지 수정 | c$   | 커서가 위치한 곳 부터 줄 끝까지 수정 |
| s, cl | 커서 위치의 한 글자를 여러 글자로 바꾸기 | cc   | 커서가 있는 줄을 교체                |
| cj    | 커서가 있는 줄과 그 다음 줄을 교체       | ck   | 커서가 있는 줄과 그 앞줄을 교체      |



### 간단한 명령

| u             | 실행 취소                                    | U               | 커서가 위치한 줄 전체 실행 취소       |
| ------------- | -------------------------------------------- | --------------- | ------------------------------------- |
| CTRL-R        | re-do                                        | p               | 버퍼의 내용 붙이기                    |
| CTRL-g        | 파일 내에서 현재 위치와 파일의 상태 보기     | %               | ), ], } 의 대응하는 짝 찾기           |
| shift-G       | 파일의 마지막으로 이동                       | 13shift-G       | 13번 라인으로 이동                    |
| /             | 입력한 내용 찾기 (뒤로 찾기)                 | ?               | 입력한 내용 찾기 (앞으로)             |
| %s/old/new/g  | 파일 전체에서 발견된 모든 것을 치환하는 경우 | :s/old/new/g    | 'old'를 'new'로 치환                  |
| %s/old/new/gc | 바꿀 때 마다 확인하기                        | 1,10s/old/new/g | 1~10번 줄 사이의 'old'를 'new'로 치환 |
| y             | 커서가 위치한 곳이나 블럭 복사               | yy              | 커서가 위치한 줄 복사                 |
| =             | 들여쓰기 자동 정렬                           |                 |                                       |

### 명령 모드

| :!command         | :!를 입력하면 외부(쉘)명령을 사용할 수 있다.             |
| ----------------- | -------------------------------------------------------- |
| :10,23 w filename | 현재 문서의 10~23라인을 filename이라는 파일명으로 저장   |
| :r filename       | filename이라는 파일을 가져와 커서 뒤에 집어넣기          |
| :set xxx          | "xxx" 옵션을 설정하는 명령                               |
| :set ic           | 대소문자를 구분하지 않는다.(Ignore case) (찾기 / 바꾸기) |
| :set hls          | 찾은 내용 강조하기(highlight) - hlsearch                 |
| :set is           | incsearch                                                |
| :set number       | 라인 넘버                                                |
| :syntax on        | 문법 강조 켜기                                           |
| :set ts=4         | 탭 사이즈 4칸으로 설정                                   |
